# Simplify Path (medium, 71).

# Вам дан абсолютный путь в Unix-подобной файловой системе, который
# всегда начинается с символа '/'. Ваша задача - преобразовать этот
# абсолютный путь в упрощённный канониченый путь.

# Правила Unix-подобной файловой системы:

# * Одинарная точка '.' обозначает текущую директорию.
# * Две точки '..' обозначают переход в родительскую директорию (предыдущую) директорию.
# * Несколько подряд идущих слэшей, например '//' или '///', считаются одним слэщем '/'.
# * Любая последовательность точек, не соответствующая правилам выше, считается допустимым
# именем директории или файла. Например, '...' и '....' являются корректными именами
# директорий или файлов.

# Упрощённый канонический путь должен соответствовать следующим правилам:
# * Путь должен начинаться с одного слэша '/'.
# * Директории в пути должны разделяться ровно одним слэшем '/'.
# * Путь не должен заканчиваться слэшем '/', если только это не корневая директория.
# * В пути не должно быть элементов '.' и '..', используемых для обозначения текущей
# или родительской директории.

# Верните упрощённый канонический путь.
