# Multiply String ( Перемножение строк ).

# Условие задачи:

# Даны два неотрицательных целых числа num1 и num2, представленных в виде строк.
# Верните произведение num1 и num2, также представленное в виде строки.

# Примечание: запрещено использовать какие-либо встроенные библиотеки для работы
# с большими числами (BigInteger) или преобразовывать входные данные непосредственно
# в целые числа.

# Пример 1:
# Вход: num1 = "2", num2 = "3"
# Выход: "6"

# Пример 2:
# Вход: num1 = "123", num2 = "456"
# Выход: "56088"

# Ограничения:
# 1 <= num1.length, num2.length <= 200
# num1 и num2 состоят только из цифр.
# Ни num1, ни num2 не содержат ведущих нулей, за исключением самого числа 0.
