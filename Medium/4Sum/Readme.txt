# 4Sum.

# Условие:

# Дан массив nums из n целых чисел. Верните массив всех уникальных четвёрок
# [nums[a], nums[b], nums[c], nums[d]] таких, что:

# 0 <= a, b, c, d < n (индексы элементов находятся в пределах массива)
# a, b, c, d - различные (все индексы уникальны, то есть используются разные элементы массива).
# nums[a] + nums[b] + nums[c] + nums[d] == target (сумма значений элементов равна целевому числу target).
# Ответ можно возвращать в любом порядке.

# Пример 1:
# Входные данные: nums = [1, 0, -1, 0, -2, 2], target = 0
# Выходные данные: [[-2, -1, 1, 2], [-2, 0, 0, 2], [-1, 0, 0, 1]]

# Пример 2:
# Входные данные: nums = [2, 2, 2, 2, 2], target = 8
# Выходные данные: [[2, 2, 2, 2]]

# Ограничения:
# 1 <= nums.length <= 200 (длина массива от 1 до 200)
# -10^9 <= nums[i] <= 10^9 (значения элементов могут быть очень большими)
# -10^9 <= target <= 10^9 (целевое число также может быть очень большим)
