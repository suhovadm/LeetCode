# Pow(x, n)

# Реализуйте функцию pow(x, n), которая вычисляет число x, возведённое в степень n (то есть x в степени n).

# Пример 1:
# вход: x = 2.00000, n = 10
# выход: 1024.00000

# Пример 2:
# вход: x = 2.10000, n = 3
# выход: 9.26100

# Пример 3:
# вход: x = 2.00000, n = -2
# выход: 0.25000
# объяснение: 2 в -2 степени = 1/2 в 2 степени = 1/4 = 0.25

# Ограничения:
# -100.0 < x < 100.0
# -2 в 31 степени больше или равно n больше или равно 2 в 31 степени -1
# n - целое число
# Либо x не равно нулю, либо n > 0
# -10 в 4 степени больше или равно x в n-ной степени больше или равно 10 в четвёртой степени.
