# Insert Interval

# Дан массив непересекающихся интерваолов intervals, где
# intervals[i] = [starti, endi] представляет начало и конец i-го интервала.
# Массив intervals отсортирован по возрастанию starti.

# Также дан интервал newInterval = [start, end], представляющий начало и конец нового интервала.

# Вставьте newInterval в массив intervals так, чтобы:

# * массив оставался отсортированным по возрастанию starti
# * интервалы не пересекались (при необходимости объедините пересекающиеся интервалы)

# Верните новый массив интервалов после вставки.
# Обратите внимание: изменять исходный массив не обязательно - можно создать новый
# и вернуть его.

# Пример 1:
# вход: intervals = [[1,3],[6,9]], newIntervals = [2,5]
# выход: [[1,5],[6,9]]

# Пример 2:
# вход: intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]], newInterval = [4,8]
# выход: [[1,2],[3,10],[12,16]]

# Объяснение:
# интервал [4,8] пересекается с [3,5], [6,7] и [8,10].

# Ограничения:
# 0 <= intervals.length <= 10⁴
# intervals[i].length == 2
# 0 <= starti <= endi <= 10⁵
# intervals отсортирован по starti по возрастанию
# newInterval.length == 2
# 0 <= start <= end <= 10⁵

# Суть задачи максимально простая:
# - у нас есть отсортированный список непересекающихся отрезков.
# - И нам дают ещё один новый интервал.
# - Нужно: вставить этот новый интервал в список.
# - При необходимости объединить (слить) пересекающиеся интервалы
# - чтобы в итоге снова был список без пересечений и отсортированный.

# [1,3] - включает числа от 1 до 3.
# [2,5] - включает числа от 2 до 5.
# Общие числа есть?
# Да: 2 и 3.
# Значит, они пересекаются.
# Пересекаются - значит накладываются или касаются.
# Не пересекаются - между ними есть промежуток.
