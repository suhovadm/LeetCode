# Spiral Matrix II.

# Учитывая положительное целое число n, создай матрицу размером n x n, заполненную элементами
# от 1 до n² в спиральном порядке.

# Пример 1:
# входные данные: n = 3
# результат: [[1,2,3],[8,9,4],[7,6,5]]

# Пример 2:
# входные данные: n = 1
# результат: [[1]]

# Ограничения:
# 1 <= n <= 20
