# Maximum Subarray

# Да целочисленный массив nums, найди подмассив с наибольшей суммой и верни эту сумму.

# Пример 1:
# Входные данные: nums = [-2,1,-3,4,-1,2-1,-5,4]
# Результат: 6
# Пояснение: подмассив [4,-1,2,1] имеет наибольшую сумму 6.

# Пример 2:
# Входные данные: nums = [1]
# Результат: 1
# Пояснение: подмассив [1] имеет наибольшую сумму 1.

# Пример 3:
# Входные данные: nums = [5,4,-1,7,8]
# Пояснение: подмассив [5,4,-1,7,8] имеет наибольшую сумму 23.

# Ограничения:
# 1 <= nums.length <= 105
# -104 <= nums[i] <= 104
