# Set Matrix Zeroes (Medium, 73).

# Дана целочисленная матрица размером m x n.
# Если какой-либо элемент равен 0, необходимо обнулить всю его строку и весь его столбец.

# Это нужно сделать на месте (in-place), то есть без использования дополнительной матрицы.

# Пример 1.
# Вход: matrix = [[1,1,1], [1,0,1], [1,1,1]]
# Выход: matrix = [[0,1,2,0], [3,4,5,2], [1,3,1,5]]

# Пример 2.
# Вход: matrix = [[0,1,2,0], [3,4,5,2], [1,3,1,5]]
# Выход: matrix = [[0,0,0,0], [0,4,5,0], [0,3,1,0]]

# Ограничения:
# m == количество строк в matrix
# n == количество столбцов в matrix
# 1 ≤ m, n ≤ 200
# -2³¹ ≤ matrix[i][j] ≤ 2³¹ - 1

# Чего мы хотим добиться?
# Если в матрице есть хотя бы один ноль - зачёркивается вся строка и весь
# столбец, где он стоит. То есть, нашёл 0 - вся его строка становится нулями,
# весь его столбец становится нулями. Крест на крест.

# Было:
# 1  1  1
# 1  0  1
# 1  1  1

# Стало:
# 1  0  1
# 0  0  0
# 1  0  1

# Итоговая идея алгоритма.
# 1. Проверить первую строку и столбец (сохранить флаги).
# 2. Использовать их как "хранилище меток".
# 3. Пройти матрицу и поставить метки.
# 4. Применить метки + отдельно обработать первую строку и столбец.
