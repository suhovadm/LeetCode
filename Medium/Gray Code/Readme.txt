# Gray Code.

# Последовательность n-битного кода Грея - это последовательность из 2n целых чисел, в которой:

# * Каждое число находится в диапазоне [0, 2n - 1],
# * Первое число равно 0,
# * Каждое число встречается не более одного раза,
# * Двоичное представление каждой пары соседних чисел отличается ровно на один бит,
# * Двоичное представление первого и последнего чисел также отличается ровно на один бит.

# Дано целое число n. Верните любую корректную последовательность n-битного кода Грея.

# Пример 1:
# Вход: n = 2
# Выход: [0, 1, 3, 2]

# Объяснение:
# двоичное представление последовательности [0, 1, 3, 2] - это [00, 01, 11, 10].

# 00 и 01 отличаются на один бит.
# 01 и 11 отличаются на один бит.
# 11 и 10 отличаются на один бит.
# 10 и 00 отличаются на один бит.

# Также корректной является последовательность [0, 2, 3, 1], её двоичное представление: [00, 10, 11, 01].
# * 00 и 10 отличаются на один бит.
# * 10 и 11 отличаются на один бит.
# * 11 и 01 отличаются на один бит.
# * 01 и 00 отличаются на один бит.

# Пример 2:
# Вход: n = 1
# Выход: [0, 1]

# Ограничения:
# * 1 ≤ n ≤ 16
