# Count and Say

# Последовательность "смотри-и-говори" - это последовательность строк и цифр, определяемая
# рекурсивной формулой:

# countAndSay(1) = "1"
# countAndSay(n) - это результат кодирования длин серий (RLE) для строки countAndSay(n - 1).

# Кодирование длин серии (RLE) - это метод сжатия строки, который работает путём замены
# последовательных одинаковых символов (повторяющихся 2 или более раз) на объединение самого
# символа и числа, обозначающего количество этих символов (длину серии). Например, чтобы сжать
# строку "3322251", мы заменяем "33" на "23", заменяем "222" на "32", заменяем "5" на "15" и
# заменяем "1" на "11". Таким образом, сжатая строка становится "23321511".

# Дано положительное целое число n, верните n-й элемент последовательности "смотри-и-говори".

# Пример 1:
# Вход: n = 4
# Выход: "1211"
# Пояснение:
# countAndSay(1) = "1"
# countAndSay(2) = RLE от "1" = "11"
# countAndSay(3) = RLE от "11" = "21"
# countAndSay(4) = RLE от "21" = "1211"

# Пример 2.
# Вход: n = 1
# Выход: "1"
