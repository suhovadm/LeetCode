Алгоритм (пример)

Вход:
nums = [-1, 2, 1, -4]
target = 1

1. Сортируем массив:
nums -> [-4, -1, 1, 2]

2. Начальное значение closest
Берём сумму первых трёх элементов:
closest = -4 + (-1) + 1 = -4

3. Внешний цикл (фиксируем первое число)
Для каждого nums[i] используем два указателя:
left = i + 1
right = конец массива

4. Два указателя (поиск пары)
Считаем:
total = nums[i] + nums[left] + nums[right]
если |total - target| < |closest - target| -> обновляем closest
если total < target -> left++
если total > target -> right--

5. Проход по примеру
При i = 0 лучший результат становится -1
При i = 1 лучший результат становится 2
Дальше проверять нечего

6. Результат
closest = 2

Как работает алгоритм?
- Сортируем массив
- Фиксируем одно число
- Двумя указателями подбираем остальные
- Запоминаем сумму, ближайшую к target
- Готово!
