Наглядный пример работы.
Возьмём массив nums = [-1, 0, 1, 2, -1, -4]

Шаг 1: Сортировка.
[-4, -1, -1, 0, 1, 2]

Шаг 2: i = 0 (nums[i] = -4)

1. Проверки пропуска дубликатов нет (i = 0)
2. -4 + (-1) + (-1) = -6 < 0 --> не break
3. -4 + 1 + 2 = -1 < 0 --> continue (переходим к следующему i)

Шаг 3: i = 1 (nums[i] = -1)

1. Пропускаем дубликаты? nums[1] == nums[0]? (-1 == -4) --> нет
2. -1 + (-1) + 0 = -2 < 0 --> не break
3. -1 + 1 + 2 = 2 > 0 --> не continue

Поиск пар:

left = 2, right = 5: -1 + (-1) + 2 = 0 [V] Нашли! Добавляем [-1, -1, 2]
Пропускаем дубликаты: нет

left = 3, right = 4
left = 3, right = 4: -1 + 0 + 1 = 0 [V] Нашли! Добавляем [-1, 0, 1]

Пропускаем дубликаты: нет

left = 4, right = 3 --> цикл завершается

Шаг 4: i = 2 (nums[i] = -1)
Пропускаем дубликаты? nums[2] == nums[1]? (-1 == -1) --> ДА! Пропускаем.

Далее поиск продолжается...
Итоговый результат: [[-1, -1, 2], [-1, 0, 1]]
