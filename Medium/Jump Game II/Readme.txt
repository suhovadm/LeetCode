# Jump Game II

# Дан массив целых чисел nums, индексированный с нуля, длины n. Изначально вы находитесь на
# позиции с индексом 0.

# Каждый элемент nums[i] обозначает максимальную длину прыжка вперёд с позиции i. Другими
# словами, если вы находитесь в позиции i, вы можете прыгнуть в любую позицию (i + j), где:

# * 0 <= j <= nums[i], и
# * i + j < n
# Ваша задача - вернуть минимальное количество прыжков, необходимое, чтобы достичь позиции n - 1.

# Гарантируется, что вы всегда можете добраться до последнего индекса.

# Пример 1:
# вход: nums = [2, 3, 1, 1, 4]
# выход: 2
# Объяснение:
# минимальное количество прыжков - 2.
# Сначала прыгаем на 1 шаг: с индекса 0 на 1, затем на 3 шага - до последнего индекса.

# Пример 2:
# вход: nums = [2, 3, 0, 1, 4]
# Выход: 2

# Ограничения:
# * 1 <= nums.length <= 10⁴
# * 0 <= nums[i] <= 1000
# * Гарантируется, что можно добраться до nums[n - 1]
