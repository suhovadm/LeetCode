# Binary Tree Postorder Traversal (Easy, 145).

# Дан корень бинарного дерева. Верните значения его узлов
# в порядке постфиксного обхода (postorder traversal).

# Пример 1:
# вход: root = [1, null, 2, 3]
# выход: [3, 2, 1]
# Пояснение: сначала левое поддерево, затем правое, затем корень.

# Пример 2:
# вход: root [1, 2, 3, 4, 5, null, 8, null, null, 6, 7, 9]
# выход: [4, 6, 7, 5, 2, 9, 8, 3, 1]

# Пример 3:
# вход: root = []
# выход: []

# Пример 4:
# root = [1]
# выход: [1]

# Ограничения:
# Количество узлов в дереве: от 0 до 100
# Значение узла: от -100 до 100

# Чего мы собственно хотим добиться?
# Мы хотим просто обойти всё дерево и записать значения узлов в правильном порядке.
# Сначала левый потомок - потом правый - потом сам узел.
# Сам узел (корень) идёт самым последним, т.е. 1 идёт самой последней.
# 3 - 2 - 1, т.е. [3, 2, 1]
