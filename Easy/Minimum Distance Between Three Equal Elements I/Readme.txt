# Minimum Distance Between Three Equal Elements I.

# Вам дан целочисленный массив nums.

# Кортеж (i, j, k) из трёх различных индексов называется хорошим,
# если nums[i] == nums[j] == nums [k].

# Расстояние хорошего кортежа определяется как
# abs(i - j) + abs(j - k) + abs(k - i),
# где abs(x) - абсолютное значение числа x.

# Верните целое число - минимально возможное расстояние среди всех хороших кортежей.
# Если хороших кортежей нет, верните -1.

# Пример 1:
# вход: nums = [1, 2, 1, 1, 3]
# выход: 6
# Объяснение: минимальное расстояние достигается для кортежа (0, 2, 3).
# Кортеж (0, 2, 3) является хорошим, потому что nums[0] == nums[2] == nums[3] == 1.
# Его расстояние: abs(0 - 2) + abs(2 - 3) + abs(3 - 0) = 2 + 1 + 3 = 6

# Пример 2:
# вход: nums = [1, 1, 2, 3, 2, 1, 2]
# выход: 8
# Объяснение: минимальное расстояние достигается для кортежа (2, 4, 6).
# Кортеж (2, 4, 6) является хорошим, потому что nums[2] == nums[4] == nums[6] == 2.
# Его расстояние: abs(2 - 4) + abs(4 - 6) + abs(6 - 2) = 2 + 2 + 4 = 8.

# Пример 3:
# вход: nums = [1]
# выход: -1
# Объяснение: хороших кортежей нет, поэтому ответ равен -1.

# Ограничения:
# 1 <= n == nums.length <= 100
# 1 <= nums[i] <= n

# Одним словом, нужно найти 3 одинаковых числа в массиве, расположенных так, чтобы
# они были как можно ближе друг к другу по индексам, и вернуть расстояние между ними.
