# Happy Number

# Напишите алгоритм, чтобы определить, является ли число n счастливым.
# Счастливое число - это число, определяемое следующим процессом:

# 1. Начиная с любого положительного целого числа, замените это число суммой
# квадратов его цифр.

# 2. Повторяйте процесс до тех пор, пока число не станет равным 1 (где оно останется),
# или пока оно не войдёт в бесконечный цикл, который не включает 1.

# 3. Числа, для которых этот процесс заканчивается на 1, являются счастливыми.

# Верните true, если n является счастливым числом, и false в противном случае.

# Пример 1:
# Вход: n = 19
# Выход: true
# Объяснение:
# 1² + 9² = 82
# 8² + 2² = 68
# 6² + 8² = 100
# 1² + 0² + 0² = 1

# Пример 2:
# Вход: n = 2
# Выход: false
