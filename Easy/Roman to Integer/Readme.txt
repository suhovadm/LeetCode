# Римские цифры представлены семью различными символами: I, V, X, L, C, D и M.
# Символ Значение:
# I 1
# V 5
# X 10
# L 50
# C 100
# D 500
# M 1000

# Например, число 2 записывается как II в римской системе счисления — просто две единицы,
# сложенные вместе. 12 записывается как XII, то есть X + II.
# Число 27 записывается как XXVII, что равно XX + V + II.

# Обычно римские цифры записываются от большего к меньшему слева направо.
# Однако цифра для четырёх пишется не как IIII.
# Вместо этого число четыре записывается как IV.
# Так как единица стоит перед пятёркой, мы вычитаем её, получая четыре.
# Тот же принцип применяется для числа девять, которое записывается как IX.
# Существует шесть случаев, когда используется вычитание:

# I может стоять перед V (5) и X (10), чтобы образовать 4 и 9.
# X может стоять перед L (50) и C (100), чтобы образовать 40 и 90.
# C может стоять перед D (500) и M (1000), чтобы образовать 400 и 900.

# Задана римская цифра, преобразуйте её в целое число.

# Пример 1:
# Ввод: s = "III"
# Вывод: 3
# Объяснение: III = 3.

# Пример 2:
# Ввод: s = "LVIII"
# Вывод: 58
# Объяснение: L = 50, V = 5, III = 3.

# Пример 3:
# Ввод: s = "MCMXCIV"
# Вывод: 1994
# Объяснение: M = 1000, CM = 900, XC = 90 и IV = 4.

# Ограничения:
# 1 <= s.length <= 15
# s содержит только символы ('I', 'V', 'X', 'L', 'C', 'D', 'M').
# Гарантируется, что s является допустимой римской цифрой в диапазоне [1, 3999].

# ---------------------------------------------------------------------------------------- #

# Дано: строка с римскими цифрами, например MCMXCIV.
# Что нужно сделать? Надо превратить эти римские цифры в обычное число, например, 1994.
# Как это сделать? У каждой римской буквы есть своё значение:
# I = 1, V = 5, X = 10 и т.д.
# Обычно просто складываем значения слева направо: III = 1+1+1 = 3.
# НО! Есть особые случаи, когда меньшая цифра стоит перед большей - тогда нужно вычитать,
# а не складывать:
# IV = 4 (5-1)
# IX = 9 (10-1)
# CM = 900 (1000-100)
# Как определить, когда вычитать?
# Смотрим на две соседние буквы.
# Если левая буква меньше правой -> вычитаем левую.
# Если левая буква больше или равна правой -> прибавляем левую.
# Пример MCMXCIV = 1994
# 1. M (1000) -> +1000
# 2. C (100) -> стоит перед M(1000), и 100 < 1000 -> вычитаем 100
# 3. M (1000) -> +1000
# 4. X (10) -> стоит перед C (100), и 10 < 100 -> вычитаем 10
# 5. C (100) -> +100
# 6. I (1) -> стоит перед V (5), и 1 < 5 -> вычитаем 1
# 7. V (5) -> +5
# Всё вместе: 1000 - 100 + 1000 - 10 + 100 - 1 + 5 = 1994.
