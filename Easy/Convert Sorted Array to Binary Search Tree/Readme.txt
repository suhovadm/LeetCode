# Convert Sorted Array to Binary Search Tree (Easy).

# Дан целочисленный массив nums, элементы которого отсортированы по возрастанию.
# Необходимо преобразовать его в сбалансированное по высоте бинарное дерево поиска (BST).

# Пример 1:
# вход: nums = [-10, -3, 0, 5, 9]
# выход: [0, -3, 9, -10, null, -5] * опечатка в исходном условии задачи! -5 нет на входе!!!
# пояснение:
# также допустим вариант:
# [0, -10, 5, null, -3, null, 9]

# Пример 2:
# вход: nums = [1, 3]
# выход: [3, 1]
# пояснение:
# допустимы оба варианта:
# [1, null, 3] и [3, 1] - оба являются сбалансированными BST.

# Ограничения:
# 1 <= nums.length <= 10⁴
# -10⁴ <= nums[i] <= 10⁴
# nums отсортирован по строго возрастающему порядку.

# То есть, наша задача, из отсортированного массива построить сбалансированное бинарное
# дерево поиска (BST). Главная цель сделать так, чтобы дерево было максимально "ровным",
# то есть: глубина слева = глубине справа, а операции поиска работали быстро.

# Пример:
# nums = [-10, -3, 0, 5, 9] * Первый вариант, в выводе будет второй вариант. Оба варианта верные!
# mid = (left + right) // 2 - это где -10 сверху!
# mid = (left + right + 1) // 2 - это где -3 сверху!
#
#         0        ← середина
#        / \
#     -10   5
#       \     \
#       -3     9
