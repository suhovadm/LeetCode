# Задача.

# Дано целое число x. Верните true, если x является палиндромом, и false в противном случае.
 
# Пример 1:
# Ввод: x = 121
# Вывод: true
# Объяснение: 121 читается как 121 слева направо и справа налево.
 
# Пример 2:
# Ввод: x = -121
# Вывод: false
# Объяснение: 
# Слева направо читается как -121. 
# Справа налево становится 121-. Поэтому это не палиндром.
 
# Пример 3:
# Ввод: x = 10
# Вывод: false
# Объяснение: Читается как 01 справа налево. Поэтому это не палиндром.
 
# Ограничения: 
# -2³¹ ≤ x ≤ 2³¹ - 1
