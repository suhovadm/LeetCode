# Complement of Base 10 Integer

# Дополнение целого числа - это целое число, которое получается путём замены всех 0 на 1
# и всех 1 на 0 в его двоичном представлении.

# Например, целое число 5 в двоичной системе представляется как "101", а его дополнение - "010",
# что соответствует целому числу 2.
# Для заданного числа n верните его дополнение.

# Пример 1:
# Вход: n = 5
# Выход: 2
# Пояснение: 5 в двоичной системе - это "101", а его дополнение в двоичной системе - "010",
# что в десятичной системе равно 2.

# Пример 2:
# Вход: n = 7
# Выход: 0
# Пояснение: 7 в двоичной системе - это "111", а его дополнение в двоичной системе - "000",
# что в десятичной системе равно 0.

# Пример 3:
# Вход: n = 10
# Выход: 5
# Пояснение: 10 в десятичной системе - это "1010", а его дополнение в двоичной системе - "0101",
# что в десятичной системе равно 5.
