# Majority Element

# Дан массив nums размером n. Верните элемент, встречающийся большинство раз.

# Элементом большинства называется элемент, который встречается в массиве более [n / 2] раз.
# Можно предположить, что элемент большинства всегда существует в массиве.

# Пример 1:
# Входные данные: nums [3,2,3]
# Результат: 3

# Пример 2:
# Входные данные: nums = [2,2,1,1,1,2,2]
# Результат: 2

# Ограничения:
# n - длина массива nums.
# 1 <= n <= 5 * 10⁴
# -10⁹ <= nums[i] <= 10⁹
# Входные данные сгенерированы так, что элемент большинства гарантированно
# будет существовать в массиве.
