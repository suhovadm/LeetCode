# Binary Tree Preorder Traversal (Easy, 144).

# Дано корневое значение бинарного дерева. Необходимо вернуть значения узлов
# в порядке прямого обхода (preorder traversal).

# Пример 1:
# вход: root = [1, null, 2, 3]
# выход: [1, 2, 3]
# Объяснение: сначала корень, затем левое поддерево, затем правое.

# Пример 2:
# вход: root = [1, 2, 3, 4, 5, null, 8, null, null, 6, 7, 9]
# выход: [1, 2, 4, 5, 6, 7, 3, 8, 9]

# Пример 3:
# вход: root = []
# выход: []

# Пример 4:
# вход: root = [1]
# выход: [1]

# Ограничения:
# Количество узлов в дереве находится в диапазоне [0, 100]
# -100 <= Node.val <= 100

# Чего мы собственно хотим добиться?
# 1. Сначала мы собираем корень.
# 2. Мы собираем всё ЛЕВОЕ поддерево. В нём сначала собираются все ЛЕВЫЕ ветви, затем правые.
# 3. Потом ПРАВОЕ поддерево. Если в ПРАВОМ поддереве есть ЛЕВЫЕ ветви, то сначала собираются они.
