# оно было бы вставлено, чтобы сохранить порядок.

# Нужно написать алгоритм со сложностью выполнения O(log n).

# Примеры выполнения:
# Вход: nums = [1,3,5,6], target = 5
# Выход: 2   (5 находится на индексе 2)
