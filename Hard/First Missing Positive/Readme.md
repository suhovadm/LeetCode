# Дан неотсортированный массив целых чисел nums. Верните наименьшее положительное число,
# которое отсутствует в массиве nums.

# Необходимо реализовать алгоритм, который работает за время O(n) и использует O(1)
# дополнительной памяти.

# Пример 1:
# Вход: nums = [1, 2, 0]
# Выход: 3
# Пояснение: все числа в диапазоне [1, 2] присутствуют в массиве.

# Пример 2:
# Вход: nums = [3, 4, -1, 1]
# Выход: 2
# Пояснение: число 1 присутствует в массиве, но 2 отсутствует.

# Пример 3:
# Вход: nums = [7, 8, 9, 11, 12]
# Выход: 1
# Пояснение: наименьшее положительное целое число 1 отсутствует.

# Ограничения:
# 1 ≤ nums.length ≤ 10⁵
# -2³¹ ≤ nums[i] ≤ 2³¹ − 1
