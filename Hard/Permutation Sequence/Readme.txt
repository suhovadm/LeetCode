# Permutation Sequence

# Множество [1, 2, 3, ..., n] содержит всего n! уникальных перестановок.

# Если перечислить и пронумеровать все перестановки по порядку, то для n = 3 получится
# следующая последовательность:

# "123"
# "132"
# "213"
# "231"
# "312"
# "321"

# Даны числа n и k, верните k-ю перестановку из этой последовательности.

# Пример 1:
# Вход: n = 3, k = 3
# Выход: "213"

# Пример 2:
# Вход: n = 4, k = 9
# Выход: "2314"

# Пример 3:
# Вход: n = 3, k = 1
# Выход: "123"

# Ограничения:
# 1 <= n <= 9
# 1 <= k <= n!
