# Candy

# Есть n детей, стоящих в ряд. Каждому ребёнку присвоен рейтинг,
# заданный в виде массива целых чисел ratings.

# Вам нужно раздать детям конфеты с соблюдением следующих условий:

# 1. Каждый ребёнок должен получить как минимум одну конфету.
# 2. Дети с более высоким рейтингом должны получать больше конфет, чем их соседи.
# Верните минимальное количество конфет, необходимое для выполнения этих условий.

# Пример 1:
# вход: ratings = [1,0,2]
# выход: 5
# Объяснение: можно раздать конфеты так: первому - 2, второму - 1, третьему - 2.

# Пример 2:
# вход: ratings = [1,2,2]
# выход: 4
# Объяснение: можно раздать конфеты так: первому - 1, второму - 2, третьему - 1.
# Третий ребёнок получает 1 конфету, так как это удовлетворяет условиям.

# Ограничения:
# n == ratings.length
# 1 ≤ n ≤ 2 * 10⁴
# 0 ≤ ratings[i] ≤ 2 * 10⁴
